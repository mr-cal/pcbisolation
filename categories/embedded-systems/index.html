Atmel ATtiny ATmega AVR Quick Code Reference

I usually forget bitwise operators, basic variables, and functions when coding in Atmel Studio for an ATtiny or ATmega project. Normally, I look at some of my old code for reference. This is my focused and consolidated version of the important building blocks. Contents I/O Registers Bitwise Operators Delays Interrupts Program Memory PWM I/O Registers DDR x - Data Direction Register - Configures each pin as an input or output. The 0th bit refers to the 0th pin of the port and the 7th bit refers to the 7th bit of the port. PORT x - Data Register - The value of the pin, used for reading input or writing output P x n - The nth bit of port x.
